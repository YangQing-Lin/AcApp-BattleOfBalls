# 球球大作战小游戏

在线访问：https://app383.acapp.acwing.com.cn:31443/

## 操作方式

WEB端：鼠标右键移动，鼠标左键攻击，Q选中火球技能后左键发射火球，F选中闪现技能后左键闪现

移动端：摇杆控制玩家移动，直接点击技能攻击
